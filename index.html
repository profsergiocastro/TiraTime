ⓘ Sobre

Time Equilibrado ajuda a montar equipes de futebol amador com equilíbrio por nota, distribuição de goleiros e sorteio rápido para usar na beira da quadra.

Os dados ficam no seu navegador usando localStorage. Não há backend e a aplicação funciona como site estático no GitHub Pages.
